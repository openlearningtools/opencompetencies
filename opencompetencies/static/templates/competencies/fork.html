{{ empty_school.name }} - Fork an existing school's competency system

{{ empty_school }} currently has an empty competency system. Rather than starting from scratch, you can choose to "fork" an existing school's competency system. This means you will start out with an exact copy of that school's current system, from which you can begin to make your own modifications.

To fork an existing school, simply click on that school. You will see a confirmation that the forking was successful, and then you will be able to view your school's new system.
